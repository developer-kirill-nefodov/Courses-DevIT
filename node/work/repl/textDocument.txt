Task1
- Откройте сеанс REPL;
- Создайте массив: const arr = [1,2,3,4,5];
- Через map умножьте каждое значение на 10 (без присвоения результата в переменную!);
- Сохраните сеанс в arr.js
- Выйдете из сеанса REPL командой .exit;
- Откройте новый сеанс REPL;
- Выведете в консоль результат от функции map с прошлого сеанса и сохраните сеанс в arr.js;

- Приглашение для ввода должно начинаться с '* ' вместо '> ', и после ввода '.exit',
