Задачи по объектам:
1. Написать функцию, которая принимает объект и возвращает массив
вида [[key, value], [key, value]]
2. Написать функцию, которая "размораживает" замороженный объект
(принимает замороженный объект, возвращает новый, содержащий свойства замороженного)
3. Написать фукцию, которая принимает массив объектов и возвращает новый
объект, содержащий все свойства переданных объектов

Синтаксис:
arrToObj(arr: object[]): object
Пример:
console.log(arrToObj([{a: 1}, {b: 2, c: 3}])); //{a: 1, b: 2, c: 3}


Object.fromEntries()Метод преобразует список пар ключ-значение в объект.
